


# 蒸馏仔 Distillation-AI-APP




> 让每一台安卓设备，都拥有**人格稳定、自带记忆**的原生端侧 AI 智能体

## 项目简介
蒸馏仔 是一款基于现代 Android 技术栈从零搭建的**原生 AI Agent 框架**。
采用自研三段式人格架构 + 本地持久记忆设计，告别套壳网页封装，
真正实现**重启不失忆、后台杀进程不丢会话、人格长期稳定**。

适合二次开发、个人作品集、端侧AI学习、商业化产品打底架构。

## 核心特性
- 🧠 三段式人格加载机制，AI 人格稳定不漂移
- 💾 Room 本地持久化记忆，重启/杀进程数据不丢失
- 🔄 智能记忆自动复苏，重启接续上一轮对话
- 🔍 关键词智能记忆检索，回溯历史聊天上下文
- 📏 上下文滑动窗口管理，避免对话超限溢出
- 🛠 模块化分层架构，易扩展、易定制、易商用

## 技术栈
- Kotlin + Jetpack Compose
- MVVM + Repository 分层架构
- Room 本地数据库
- Hilt 依赖注入
- 自研三段式认知基座
