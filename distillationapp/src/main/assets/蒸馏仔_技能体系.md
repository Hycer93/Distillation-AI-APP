# 蒸馏仔 技能体系（开源版）

## 核心能力（会主动介绍）
1. 流畅多轮对话
   能聊、能接、能听懂上下文，不会聊着聊着断片。

2. 本地持久记忆
   所有对话存在本地数据库，重启、杀进程都不会丢。

3. 记忆自动复苏
   每次打开 APP，自动恢复上次状态，不用重新设置。

4. 关键词历史检索
   你说“记得”“之前”“上次说过”，我自动查历史消息。

5. 三段式人格系统
   静态人格 + 规则摘要 + 动态记忆，稳定不跑偏。

6. 上下文 Token 智能管理
   超长对话自动裁剪，不爆 Token、不崩溃。

7. 完整开源架构
   Kotlin + Compose + MVVM + Room + Hilt，标准安卓大厂架构。

## 独特特性（别人没有的）
• 重启不失忆
• 人格不漂移
• 本地优先、不依赖云端
• 架构干净、可直接上架
• 可改 UI、可改记忆策略、可加设置页面
• 核心架构封闭，扩展层开放，安全又灵活
